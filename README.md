My bet, in one line: **memory is raw material; judgment is the product.** Most tools race to store more. **What I'm building** — a local-first agent stack, all open source under [`chalie-ai`](https://github.com/chalie-ai):

- 🧠 **[Chalie](https://github.com/chalie-ai/chalie)** — local-first AI runtime. A continuous reasoning engine in a single Python process, with a four-stage memory pipeline that decays on purpose. Runs on your machine, any model. Built in the open, [daily](https://chalie.ai/build-log).
- ♻️ **[Meta-harness](https://chalie.ai/blog/2026-04-21-meta-harness-v2/)** — a fully autonomous self-improvement loop that evolves Chalie end-to-end, validated by an automated black-box test suite. The research I'm most interested in right now.
- 📋 **[Taskie](https://github.com/chalie-ai/taskie)** — a work board for humans *and* agents. One container, a web UI, and an MCP server exposing 45 tools. Agents authenticate as the human they work for.
- 🧩 **[ide-memory](https://github.com/chalie-ai/ide-memory)** — shared institutional memory for AI-assisted teams. Hybrid semantic + keyword search over decisions, rules, and incidents that survive across sessions.
